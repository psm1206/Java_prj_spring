# Java_spring
## 한국소프트웨어산업협회 '빅데이터 개발자 교육'에서 진행한 중간 프로젝트
 두 군데의 사이트에서 웹 크롤링을 하여 데이터를 수집하고 Oracle DB에 저장했다.
 이 데이터를 이용해 웹 화면에 배치해주었다.

 우선 서블릿과 JSP를 이용해 완성을 한 후, 이를 Spring 4.x으로 바꿔서 재구현해보았다.
